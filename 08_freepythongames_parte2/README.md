# Free Python Games 

Os jogos a seguir são versões dos jogos presentes no
[Free Python Games](https://grantjenks.com/docs/freegames/)(FPG).
adaptados e detalhados para uso em sala de aula.

- [Cobra](cobra.md)
- [Campo minado](campo_minado.md)

Como essas versões não estão tão funcionais como as versões do FPG e,
por outro lado, será necessário a instalação do pacote `freegames`
para que esses jogos possam ser executados, os alunos deverão executar os
jogos deste pacote para entenderem o funcionamento esperado, antes de fazer 
alterações.

Algumas dessas alterações terão por propósito deixar o jogo tal qual está
no FPG. Os alunos deverão começar por essas, que estarão marcado por TODO
no código. Outras alterações são para  evoluir sob diversos aspectos ou 
expandir o entendimento dos estudantes, que estarão na forma de exercícios
no topo dos arquivos.

A documentação do Free Python Games fornece um
[Quickstart](https://grantjenks.com/docs/freegames/#quickstart).
